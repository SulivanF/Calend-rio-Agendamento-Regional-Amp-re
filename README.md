# Agenda Regional Ampére

Um sistema de agenda simples, responsivo e focado na organização de eventos para a região de Ampére - PR. O projeto utiliza `localStorage` para persistir os dados diretamente no navegador do usuário, sem a necessidade de um banco de dados externo.

## 🚀 Funcionalidades

- **Calendário Interativo:** Navegação entre meses para visualização de datas.
- **Destaque de Hoje:** Visualização rápida do dia atual.
- **Gestão de Eventos:** Adicione, edite e remova compromissos com facilidade.
- **Persistência de Dados:** Seus eventos são salvos automaticamente no navegador.
- **Interface Dark:** Design moderno e confortável para uso prolongado.

## 🛠️ Tecnologias Utilizadas

- [Tailwind CSS](https://tailwindcss.com/): Para estilização rápida e responsiva.
- [Lucide Icons](https://lucide.dev/): Biblioteca de ícones moderna e leve.
- **JavaScript (Vanilla):** Lógica de manipulação de datas e persistência (`localStorage`).
